🔒 Datenschutzhinweis

Diese App speichert deinen Lernfortschritt nur lokal auf deinem Gerät. Die gemeinsame Karteidatenbank wird über den schulischen ByCS Drive geteilt. Es werden keine Daten an externe Server gesendet.

Weitere Infos findest du unter "Einstellungen → Datenschutz".
